## 開発環境

- 開発言語：Python3.14
- フレームワーク：FastAPI
  - `python3.14 -m venv env`
  - `pip install fastapi uvicorn`
- vscode を使用しています(ツールの使用を強制していませんが、参考までに!)
- 開発に使用している OS：Pop!OS **(かなりおすすめ!)**

## どういうサイトにしたいか

ぽこピーのファンサイトを趣味で作りたいな〜って思ってます<br>
偉大な本家様に迷惑がかからない範囲で楽しくできたらな〜って思います

ぽこピー様は数々の動画で名言を放っており、その名言はどこの動画で放ったのかを見れることで、<br>
新たな里の民を増やせると思っています(布教活動)

## ゴールイメージ

Youtube とかニコ動みたいに動画が一覧化されるみたいな感じで、見やすいサイトにしてプルリクを捌いて運用できるようにしたいな〜(この辺もう少し固める)
